HTML Basics - Topics:

  • Introduction to HTML

    HTML Introduction

    HTML (Hypertext Markup Language) is the standard markup language used to create web pages.

    HTML structures the content and elements of a web page. Elements are defined using tags like <h1>, <p>, <a>, and more.

  • HTML Tags and Attributes

    HTML Tags

    Tags are the building blocks of an HTML document. Common tags include:

    • <p> for paragraphs
    • <h1> to <h6> for headings
    • <a> for links
    • <img> for images

    Tags can include attributes that define specific properties. For example, <a href="URL"> defines the destination of a link.

  • Creating Links and Images

    Links & Images

    Links: Links are created using the <a> tag. For example:

    <a href="https://example.com">Visit Example</a>

    Images: Images are embedded using the <img> tag:

    <img src="image.jpg" alt="Description">

CSS Basics - Topics:

  • Introduction to CSS

    CSS Basics

    CSS (Cascading Style Sheets) is used to style the layout and appearance of web pages. It defines how HTML elements are displayed.

    Common styles include setting colors, fonts, margins, paddings, and positioning of elements.

  • CSS Selectors and Properties

    Selectors & Properties

    Selectors target HTML elements to apply styles. For example, p { color: red; } sets the text color of paragraphs to red.

    There are different types of selectors: element selectors, class selectors (.classname), and ID selectors (#id).

  • CSS Layout and Positioning

    Layout & Positioning

    CSS allows you to arrange elements in different ways, such as using Flexbox or Grid. You can also position elements with the position property (static, relative, absolute, fixed).
